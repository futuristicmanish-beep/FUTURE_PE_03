# FUTURE_PE_03 — AI SEO Blog & Content Cluster Generator for Business Websites

## Overview
This repository contains Task 3 of the Future Interns Prompt Engineering internship: generating an SEO-optimized blog post and supporting content cluster strategy for a local business using AI.

## Details
- **Business:** Manish Gentora Cafe, Erode
- **Tool Used:** Claude AI
- **Deliverable:** SEO blog outline, full long-form blog post, 5-post content cluster strategy, and hyper-local SEO adaptation

## Contents
- `prompts/` — all prompts used
- `outputs/` — blog outline, full blog post, content cluster strategy, local SEO variant
- `docs/Prompt_Engineering_Report_Task3.docx` — full task report
- `screenshots/` — process screenshots

## Skills Demonstrated
- AI-driven SEO content creation
- Keyword intent mapping
- Content cluster and internal linking strategy
- Local/hyper-local SEO adaptation

## Key Learnings
- Chaining prompts (outline → full post → cluster → local variant) produces more consistent, on-strategy content than isolated generation.
- Structuring content to be helpful before promotional serves both SEO and reader trust.
- Hyper-local SEO requires genuine local specificity, not generic city-wide language.
